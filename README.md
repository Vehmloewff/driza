## Installation

## Usage

```js
const todo = require('todo');

todo(options);
```

## API

### todo(options: object) => void

Valid options are:

-   `todoCreate: boolean` Marks the ne project as a `todo`. Deafult is `false`
-   `delay: number` The amount of time to delay the cone. Default is `0`
